Technical Specifications

  • Tech Stack: HTML5, CSS3, Vanilla JavaScript (ES6+).

API & Rate Limits

  • Data Source: CoinGecko Public API (Free Tier).
  • Rate Limit: ~10-30 requests/minute per IP address.
  • Usage Advice: Ideal for personal portfolios, blogs, or Notion pages. High-traffic sites may require caching.
  • Auth: No API Key required for standard usage.
